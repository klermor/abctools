List All Available Chrome Webfonts

Instructions:  

1) Click "Get the font list from the browser"

You may be prompted by the browser to give permission for the operation.

The List of Font Names area will be filled with the names of all fonts available in Chrome.

2) Click "Copy font name list to the clipboard" to copy the list of font names to the system clipboard for pasting into other applications.

3) Click "Save font name list to a file" to save the list of font names to a file.

The font names in the list can be used in %%titlefont and other ABC font directives when running the ABC Transcription Tools in Chrome.
